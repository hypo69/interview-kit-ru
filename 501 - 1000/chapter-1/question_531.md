### `question_131.md`

**Вопрос 131.** Какой алгоритм сортировки реализован в представленном ниже коде Python?

```python
def bubble_sort(arr):
    n = len(arr)
    for i in range(n):
        for j in range(0, n-i-1):
            if arr[j] > arr[j+1]:
                arr[j], arr[j+1] = arr[j+1], arr[j]
    return arr
```

-   A. Сортировка вставками (Insertion Sort)
-   B. Сортировка слиянием (Merge Sort)
-   C. Пузырьковая сортировка (Bubble Sort)
-   D. Быстрая сортировка (Quick Sort)

**Правильный ответ: C**

**Объяснение:**

Представленный код реализует алгоритм пузырьковой сортировки (Bubble Sort).

*   **Как работает Bubble Sort:**
    1.  Алгоритм многократно проходит по массиву, сравнивая соседние элементы.
    2.  Если текущий элемент больше следующего, они меняются местами.
    3.  Элементы постепенно "всплывают" к концу массива, как пузырьки в воде, отсюда и название.
    4.  После каждой итерации самый большой элемент из неотсортированной части массива оказывается в конце.
    5.  Проходы по массиву продолжаются до тех пор, пока массив не будет отсортирован.
*   **Анализ кода:**
    *   Внешний цикл `for i in range(n)` управляет количеством проходов по массиву.
    *   Внутренний цикл `for j in range(0, n-i-1)` сравнивает соседние элементы и меняет их местами, если необходимо.
    *  Условие `if arr[j] > arr[j+1]` проверяет, нужно ли менять местами соседние элементы.
    *   `arr[j], arr[j+1] = arr[j+1], arr[j]` меняет местами два соседних элемента.

**Разбор вариантов:**
*   **A. Сортировка вставками (Insertion Sort):** Неправильно. Сортировка вставками работает иначе.
*   **B. Сортировка слиянием (Merge Sort):** Неправильно. Сортировка слиянием является рекурсивным алгоритмом.
*   **C. Пузырьковая сортировка (Bubble Sort):** Правильно. Это описание алгоритма.
*   **D. Быстрая сортировка (Quick Sort):** Неправильно. Quick Sort использует разделение и рекурсию.

**В результате:**

*   Код наглядно демонстрирует классический алгоритм сортировки, называемый "пузырьковой сортировкой".
*   Пузырьковая сортировка - один из самых простых в понимании алгоритмов сортировки, но не эффективен для больших наборов данных.

Таким образом, правильным ответом является **C. Пузырьковая сортировка (Bubble Sort)**.

### `question_132.md`

**Вопрос 132.** Какой принцип работы лежит в основе алгоритма сортировки вставками (Insertion Sort)?

-   A.  Разбиение массива на подмассивы и их рекурсивная сортировка.
-   B.  Поиск минимального элемента и его перемещение в начало неотсортированной части.
-   C.  Поочередное сравнение соседних элементов и их обмен при необходимости.
-   D.  Построение отсортированного массива путем вставки элементов из исходного массива в правильную позицию.

**Правильный ответ: D**

**Объяснение:**

Сортировка вставками работает, строя отсортированную часть массива по одному элементу за раз. На каждой итерации элемент из неотсортированной части массива выбирается и вставляется на правильную позицию в уже отсортированной части.

*   **Принцип работы:**
    1.  Начинаем с первого элемента массива (он считается отсортированным).
    2.  Проходим по неотсортированной части массива.
    3.  Выбираем текущий элемент из неотсортированной части.
    4.  Сравниваем текущий элемент с элементами в отсортированной части, сдвигая их вправо, пока не найдем правильное место для текущего элемента.
    5.  Вставляем текущий элемент на найденную позицию.

**Разбор вариантов:**
*   **A. Разбиение массива на подмассивы и их рекурсивная сортировка:** Неправильно. Это принцип сортировки слиянием и быстрой сортировки.
*   **B. Поиск минимального элемента и его перемещение в начало неотсортированной части:** Неправильно. Это принцип сортировки выбором.
*  **C. Поочередное сравнение соседних элементов и их обмен при необходимости:** Неправильно. Это принцип пузырьковой сортировки.
*   **D. Построение отсортированного массива путем вставки элементов из исходного массива в правильную позицию:** Правильно. Это точное описание работы сортировки вставками.

**В результате:**
*   Сортировка вставками эффективна для небольших массивов и частично отсортированных данных.
*   Это простой алгоритм с реализацией "на месте" (in place).

Таким образом, правильным ответом является **D. Построение отсортированного массива путем вставки элементов из исходного массива в правильную позицию.**

### `question_133.md`

**Вопрос 133.** Что является ключевой характеристикой алгоритма сортировки слиянием (Merge Sort)?

-   A. Он работает путем многократного выбора минимального элемента из неотсортированной части массива.
-   B. Он работает путем сравнения и обмена соседних элементов.
-   C. Он является рекурсивным алгоритмом, который разделяет массив на подмассивы и затем их сливает.
-   D. Он является алгоритмом сортировки на месте, не требующим дополнительной памяти.

**Правильный ответ: C**

**Объяснение:**

Сортировка слиянием (Merge Sort) является рекурсивным алгоритмом сортировки, который использует принцип "разделяй и властвуй". Он разбивает массив на более мелкие подмассивы до тех пор, пока каждый подмассив не будет содержать только один элемент (который считается отсортированным). Затем отсортированные подмассивы сливаются обратно в большие отсортированные массивы, пока не получится один полностью отсортированный массив.

*   **Принцип работы:**
    1.  Разделить массив на две половины (рекурсивно).
    2.  Рекурсивно отсортировать каждую половину.
    3.  Слить отсортированные половины в один отсортированный массив.

**Разбор вариантов:**
*   **A. Он работает путем многократного выбора минимального элемента из неотсортированной части массива:** Неправильно. Это принцип сортировки выбором.
*   **B. Он работает путем сравнения и обмена соседних элементов:** Неправильно. Это принцип пузырьковой сортировки.
*   **C. Он является рекурсивным алгоритмом, который разделяет массив на подмассивы и затем их сливает:** Правильно. Это ключевая характеристика сортировки слиянием.
*  **D. Он является алгоритмом сортировки на месте, не требующим дополнительной памяти:** Неправильно. Сортировка слиянием требует дополнительную память для слияния подмассивов.

**В результате:**
*   Сортировка слиянием является эффективным алгоритмом сортировки со стабильной производительностью O(n log n) во всех случаях.
*   Это не алгоритм "на месте", так как он использует дополнительную память.

Таким образом, правильным ответом является **C. Он является рекурсивным алгоритмом, который разделяет массив на подмассивы и затем их сливает.**

### `question_134.md`

**Вопрос 134.** В чем заключается основной принцип работы алгоритма быстрой сортировки (Quick Sort)?

-   A.  Сравнение и обмен соседних элементов, постепенно перемещая большие элементы в конец массива.
-  B. Разбиение массива на подмассивы относительно опорного элемента и рекурсивная сортировка подмассивов.
-   C. Выбор минимального элемента из неотсортированной части массива и его перемещение в начало.
-  D. Построение отсортированного массива путем вставки элементов из исходного массива в правильную позицию.

**Правильный ответ: B**

**Объяснение:**

Быстрая сортировка (Quick Sort) является эффективным алгоритмом сортировки, работающим по принципу "разделяй и властвуй". Он выбирает один элемент из массива в качестве "опорного" (pivot) и разбивает массив на две части относительно этого элемента: подмассив элементов, меньших опорного, и подмассив элементов, больших опорного. Затем рекурсивно сортирует эти подмассивы.

*   **Принцип работы:**
    1.  Выбрать опорный элемент (pivot) из массива.
    2.  Разделить массив на два подмассива: элементы, меньшие опорного, и элементы, большие опорного.
    3.  Рекурсивно отсортировать каждый из подмассивов.
    4.  Скомбинировать отсортированные подмассивы (не требует дополнительной операции слияния).

**Разбор вариантов:**
*   **A. Сравнение и обмен соседних элементов, постепенно перемещая большие элементы в конец массива.:** Неправильно. Это принцип пузырьковой сортировки.
*   **B. Разбиение массива на подмассивы относительно опорного элемента и рекурсивная сортировка подмассивов:** Правильно. Это точное описание принципа быстрой сортировки.
*   **C. Выбор минимального элемента из неотсортированной части массива и его перемещение в начало:** Неправильно. Это принцип сортировки выбором.
*  **D. Построение отсортированного массива путем вставки элементов из исходного массива в правильную позицию:** Неправильно. Это принцип сортировки вставками.

**В результате:**
*  Быстрая сортировка является эффективным алгоритмом с хорошей средней производительностью O(n log n).
*  В худшем случае производительность может быть O(n^2), если опорный элемент выбран неудачно.

Таким образом, правильным ответом является **B. Разбиение массива на подмассивы относительно опорного элемента и рекурсивная сортировка подмассивов.**
