### `question_148.md`

**Вопрос 48.** В Python, как ключевое слово `global` влияет на использование переменных внутри функций?

A. Оно создает новую переменную в глобальной области видимости.
B. Оно делает переменную внутри функции доступной только для чтения.
C. Оно позволяет функции изменять значение глобальной переменной.
D. Оно ограничивает переменную локальной областью видимости внутри функции.

**Правильный ответ: C**

**Объяснение:**

Ключевое слово `global` используется внутри функции для того, чтобы указать, что переменная, к которой обращается функция, является глобальной переменной, определенной вне функции.

*   **Вариант A** не верен: `global` не создает новую переменную.
*   **Вариант B** не верен: `global` не ограничивает доступ только для чтения.
*   **Вариант C** верен: `global` позволяет изменять значение глобальной переменной.
*   **Вариант D** не верен: `global` дает доступ к внешней переменной, а не ограничивает область видимости.

**Как работает `global`:**

1.  По умолчанию, при присвоении значения переменной внутри функции, Python создаёт локальную переменную, даже если есть глобальная переменная с таким же именем.
2.  Чтобы изменить значение глобальной переменной внутри функции, её нужно объявить с ключевым словом `global` внутри функции.
3.  После объявления переменной как `global`, функция может изменять эту переменную, влияя на значение переменной в глобальной области видимости.

**Пример:**

```python
global_var: int = 10 # Глобальная переменная
def modify_global():
    global global_var  # Объявляем, что global_var это глобальная переменная
    global_var = 20 # Меняем значение глобальной переменной
    print(f"Значение переменной внутри функции: {global_var}")

modify_global()
print(f"Значение переменной вне функции: {global_var}")

# Вывод:
# Значение переменной внутри функции: 20
# Значение переменной вне функции: 20
```

**В результате:**

*   Функция `modify_global` изменяет глобальную переменную `global_var`, так как она объявлена как `global`.

Таким образом, **вариант C** является правильным.
